## 要件と方針

### 1. 綿毛スライダー分離（アンビエントを手から独立）
現状 `flow`/`scale`/`react` が手と綿毛で共有＝操作しにくい。分離:
- 綿毛: `ambient`(密度・既定0.30) / `ambFlow`(流速) / `ambScale`(場スケール) / `ambReact`(明滅=拍flash強度)
- 手: `recruit` / `cohesion` / `react`(手の音反応) / `pace`
breathAt は `ambReact`、formAt は `react` を使うよう分離。draw の ambient flash は ambReact 連動。

### 2. 綿毛の流れモード（切替式 + 範囲スライダー）
modeGroup `flow` = ['Directional','Radial','Wander']。+ `spread` スライダー(範囲/半径)。
- Directional(現状): inflow edge で reseed・flowAngle 沿いに移流。
- Radial: 中心域(半径小)に spawn → **外向き放射**に移流。`spread` で最大半径。半径超で中心へ reseed＝同心円湧出。
- Wander: 流れ角を **clock+hash で緩やかに時間変化**(Math.random不可・決定論)。spawn 散らす。「出現箇所・流れが時々変わる」。
`_ambientPos` と ambient 移流をモード分岐。半径/向きは粒子位置(radial)・clock(wander)から決定論的に。

### 3. タイミング = ゆっくり（出現10秒以上）
`purposeMakerChoreo.js` DURATIONS: gather 2.6→**10** / hold 2.4→**3** / disperse 2.2→**5** / gap 0.5→**1**。
STATION 7.7→**19**。CYCLE(5)=**95**。choreo & scene テストの時間点を新タイムラインへ全面更新。
formAt の line/sheet/GHOLD のg配分は不変(gが10sに引き伸ばされるだけ)。pace で更にライブ調整可。

### 4. 5本の曲線（線の段階＝動画と同一）
線の局面で、リクルート粒子を **5本の曲線ストランド**に組織化（手首基点から扇状）。各ストランドが指へ連結。
- 各手に **wrist 基点**＋**5本の指先アンカー**(u,v、cloud から測ってハードコード or init時導出)を定義。
- 粒子をハッシュで5群に割当(`_h(i*?)%5`)。線相(F.line高)では群kの粒子を wrist→指先アンカーk の**曲線(二次ベジェ)**
  上に配置(進度=gp)。曲線の曲率で「曲線」感。→ 帯相で広がり → 収束相で実 hand target へ blend。
- 5群と実targetの不一致は、収束weightで curvePos→targetPos を lerp して継ぎ目なく。

### 5. ゆらゆら引っ張り（枠外から引かれる波打ち＋抗って前進）
5本の曲線に **進行方向の横波**(perp変位 = sin(curveParam*freq - t*spd)*amp、ampはF.lineで増減)。
wrist 側(枠外寄り)へ引く張力に**抗って前進**(net前進=gp上昇、pull=後方力+波)。その後 **帯にばらけ**(sheet相)→手。

### 6. 有機的な粘り・意志（最重要）
収束を「滑らかな機械スナップ」から「抵抗を越え粘着して寄り集まる」へ。4要素:
- **抵抗場**: 収束に**抗うノイズ場**(粒を target から逸らす)。pull がこれを越えて初めて寄る＝苦労。
- **局所粘着(粘り)**: 粗い密度グリッド(32x32)を粒子位置から毎フレーム構築→粒は高密度セルへも引かれる
  ＝表面張力的に塊で寄る(各独立スナップでなく)。
- **逡巡(意志)**: 粒子毎の確率的ためらい(hash+timeで停滞→突進)。一様でなく個々が苦労。
- **粘性**: 速度減衰でグルーピィに。
組合せ=抵抗に抗い・塊で粘り・個々がためらいつつ・最後は意志で手へ凝集。10秒かけて有機的に。
